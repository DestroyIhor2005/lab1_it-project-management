\# CryptoTrack MVP — Моніторинг цін криптовалют



\## Про проєкт

Це фронтенд-додаток, призначений для відстеження курсу цифрових активів у реальному часі. 



\*\*Ідея MVP:\*\* Створити простий та швидкий інтерфейс для моніторингу вартості обраних токенів (BTC, ETH, INK) через підключення до публічних API.



\## Стек технологій

\- \*\*Frontend:\*\* HTML, CSS, JavaScript (React)

\- \*\*API:\*\* CoinGecko API

\- \*\*Версіонування:\*\* Git / GitHub



\## Структура папок

\- `src/` — вихідний код додатку (логіка та компоненти).

\- `docs/` — проєктна документація та звіти до лабораторних робіт.



\## Інструкція з запуску

1\. Скопіювати посилання на репозиторій із сайту GitHub.

2\. Виконати команду `git clone \[URL-адреса\_репозиторію]` у терміналі.

3\. Перейти у папку проєкту та відкрити `index.html` у браузері.
